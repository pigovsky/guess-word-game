# Правила гри:

1. Коли ви заходите у гру на вашому балансі 0 очків.
2. Ви повинні вибрати одне питання із запропонованих.
3. Далі потрібно написати відповідь на питання.
4. Вгадавши повністю слово без помилки, ви отримаєте на рахунок 100 очків.
5. Вгадавши певні літери слова, ви отримуєте за кожну букву по 1 очку.
6. Ввівши не правильно букву, з вашого рахунку зніметься 1 очко.
