Правила гри:
1. За вгадане слово ви отримуєте +100 балів
2. Якщо ви зробили помилку у відгадуванні слова -100 балів
3. За вгадану букву ви отримуєте +1 бал
4. Якщо ви не вгадали букву -1 бал
5. Початковий рахунок складає 0 балів
6. На початку гри, коли рахунок становить 0 балів, при помилковому введенні букви/слова, 
знімаються бали, тобто рахунок може мати мінусове значення.
